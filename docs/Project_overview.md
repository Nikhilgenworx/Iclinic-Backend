# Project Overview

## Problem Statement

Healthcare clinics frequently depend on manual front-desk operations.

These processes introduce:

- Long wait times
- Scheduling errors
- Increased staff workload
- Poor patient experience

The objective of iClinic is to automate these operations using AI.

---

## Actors

### Patient

Can:

- Register
- Login
- Book appointments
- Reschedule appointments
- Cancel appointments
- Use voice assistant

### Front Desk

Can:

- Manage doctors
- Manage schedules
- View appointments

### Administrator

Can:

- View audit logs
- Monitor system activity
