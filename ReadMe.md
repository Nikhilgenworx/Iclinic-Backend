# iClinic AI Front Desk Assistant

## Overview

iClinic AI Front Desk Assistant is an AI-powered healthcare appointment management platform that automates clinic front-desk operations through chat and voice interactions.

The system allows patients to:

- Book appointments
- Reschedule appointments
- Cancel appointments
- Ask clinic-related questions
- Receive confirmations through email

The platform provides a Front Desk Dashboard for managing doctors, schedules, and appointments.
